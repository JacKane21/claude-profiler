# Contributing

Thanks for your interest in contributing!

## Quick Start
1. Fork the repo and create a feature branch.
2. Make your change and keep commits focused.
3. Run the checks below.
4. Open a PR with a clear description of the change and why it matters.

## Development
- Build: `cargo build`
- Run locally: `cargo run`
- Format: `cargo fmt`
- Lint: `cargo clippy --all-targets`
- Tests: `cargo test`

## Docs
- Update `README.md` when behavior, profiles, or configuration changes.
- Update `SECURITY.md` if reporting channels or support policy changes.
- Use placeholder values for API keys and tokens.

## Reporting Issues
Please include:
- OS and Rust version
- Steps to reproduce
- Expected vs. actual behavior
- Logs or screenshots when applicable
