# Contributing

Thanks for your interest in contributing!

## Quick Start
1. Fork the repo and create a feature branch.
2. Run the app locally to validate changes.
3. Open a PR with a clear description of the change.

## Development
- Format: `cargo fmt`
- Check: `cargo check`
- Tests: `cargo test` (if/when tests exist)

## Reporting Issues
Please include:
- OS and Rust version
- Steps to reproduce
- Expected vs. actual behavior
- Logs or screenshots when applicable
